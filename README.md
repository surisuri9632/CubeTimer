# CubeTimer
큐브 다이머 입니다.

C# 큐브 타이머 입니다.

큐브 로직에 관련된 내용은 빠진 그냥 간단한 스톱워치라 보면 됩니다.

리스트 박스에 저장되는데 저장된 값은 내장된 SQLite의 DB에 저장이 됩니다.

스페이스 키로 동작됩니다. 

스페이스 누르면 시작하고 그 다음에 정지되면서 저장하고 

한번더 누르면 초기 상태로 돌아갑니다.
